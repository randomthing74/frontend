🔬 Tugas Akhir - Robot Terapi Rehabilitasi Lengan Pasien Stroke

📋 Tentang Sistem Ini

RehabSys adalah robot terapi rehabilitasi lengan yang dirancang khusus untuk membantu pemulihan pasien stroke. Sistem ini mengintegrasikan Arduino Mega 2560 WiFi sebagai pengendali utama, ESP32 sebagai jembatan komunikasi nirkabel, serta berbagai sensor dan aktuator untuk menciptakan terapi yang adaptif dan terpantau secara real-time.

🔬 Sensor & Aktuator:
• Sensor EMG (Elektromiografi) mendeteksi sinyal listrik otot pasien untuk mengukur kekuatan kontraksi.
• Potensiometer dipasang pada setiap sendi lengan untuk mengukur sudut gerakan secara presisi.
• Motor Stepper NEMA 23 dengan driver DM542 memberikan bantuan gerakan dengan torsi yang kuat dan terkontrol.

🧠 Kontrol Fuzzy Logic:
Sistem menggunakan logika Fuzzy Logic untuk menentukan tingkat bantuan motor. Input dari sensor EMG diklasifikasikan menjadi tiga kategori: Lemah, Sedang, atau Kuat. Berdasarkan klasifikasi tersebut, motor stepper memberikan bantuan penuh, sebagian, atau minimal, sehingga terapi selalu sesuai dengan kemampuan aktual pasien dan mencegah kelelahan berlebih.

🌐 Internet of Things (IoT):
Data sensor dan status terapi dikirim secara nirkabel melalui ESP32 ke Cloudflare Worker dan disimpan di Cloudflare KV Storage. Website monitoring (frontend) menampilkan sudut lengan dan sinyal EMG secara real-time tanpa perlu instalasi aplikasi tambahan. Tenaga medis atau keluarga dapat memantau perkembangan pasien dari mana saja.

📊 Fitur Unggulan:
• Penyimpanan data pasien (nama, umur, alamat) terintegrasi.
• Tombol Mulai Terapi yang langsung mengirim perintah ke robot.
• Monitoring real-time sudut dan sinyal EMG di halaman web.
• Download data historis dalam format Excel dengan pilihan rentang tanggal.
• Sidebar navigasi yang intuitif untuk mengakses About Us, Hasil Monitoring, dan Mulai Terapi.

🏥 Manfaat Rehabilitasi:
Dengan pendekatan ini, terapi menjadi lebih personal, aman, dan terukur. Pasien dapat berlatih secara mandiri dengan pengawasan minimal, sementara tenaga medis dapat mengevaluasi kemajuan berdasarkan data objektif. Sistem ini diharapkan mampu mempercepat proses pemulihan motorik lengan pasca-stroke.

FUZZY LOGIC CONTROL

Sistem Kendali Cerdas untuk Terapi

📖 Penjelasan Fuzzy

Sistem ini menggunakan Fuzzy Logic untuk menentukan tingkat bantuan motor stepper berdasarkan kekuatan otot pasien yang diukur melalui sensor EMG.

Input: Sinyal EMG (0–1023)
Output: Persentase bantuan motor (30% – 80%)

Aturan Fuzzy:

  • Jika EMG Lemah (0–400) → Bantuan 80% (motor cepat)
  • Jika EMG Sedang (400–700) → Bantuan 50%
  • Jika EMG Kuat (>700) → Bantuan 30% (motor lambat)

Dengan pendekatan ini, terapi selalu disesuaikan dengan kemampuan aktual pasien.
